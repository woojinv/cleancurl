# cleancurl

`cleancurl` is a command-line utility for macOS that takes a messy `curl` command, redacts the `Bearer` token, removes the `Cookie` header, and automatically copies the cleaned-up version to your clipboard.

This is especially useful for safely sharing or storing `curl` commands generated by Chrome Developer Tools or Postman without leaking sensitive session data.

## Prerequisites

- **macOS**: This script uses Mac's native `pbcopy` and `pbpaste` commands for clipboard management.
- **Python 3**: Comes pre-installed on most modern Macs.

## Installation

1. **Clone the repository** (or download the `cleancurl` file):

   ```bash
   git clone https://github.com/yourusername/your-repo-name.git
   cd your-repo-name
   ```

2. **Create the script** (if you aren't cloning a repo):
   Create a file named `cleancurl` and paste the script code into it.

3. **Make the script executable**:

   ```bash
   chmod +x cleancurl
   ```

4. **Move it to your PATH** (so you can run it from anywhere):
   ```bash
   sudo mv cleancurl /usr/local/bin/
   ```

## Usage

There are two ways to use `cleancurl`:

### 1. The Clipboard Method (Recommended)

Because massive `curl` commands with single and double quotes can be hard to pass directly into the terminal without shell escaping issues, the easiest way to use this tool is to pipe your clipboard directly into it.

1. Copy the dirty `curl` command to your clipboard (e.g., from Chrome DevTools: _Right-Click -> Copy -> Copy as cURL_).
2. Run the following in your terminal:
   ```bash
   pbpaste | cleancurl
   ```
3. A sanitized version of the `curl` command is now instantly on your clipboard, ready to be pasted.

### 2. The Argument Method

You can also pass a `curl` command directly as an argument wrapped in quotes. _(Note: Watch out for unescaped single quotes in your payload!)_

```bash
cleancurl "curl --location 'https://api.example.com' --header 'Authorization: Bearer my-secret-token' --header 'Cookie: session=12345'"
```

## What it does

**Before (Input):**

```bash
curl --location 'https://api.example.com/data' \
--header 'Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR...' \
--header 'Cookie: _abck=385C1838D518E5; bm_sz=F205D311...' \
--data ''
```

**After (On your clipboard):**

```bash
curl --location 'https://api.example.com/data' \
--header 'Authorization: Bearer [REDACTED]' \
--data ''
```

## The Code

If you just need the script, here is the Python code:

```python
#!/usr/bin/env python3
import sys
import re
import subprocess

def clean_curl(curl_cmd):
    # 1. Redact the Bearer Token
    curl_cmd = re.sub(r"(Authorization:\s*Bearer\s+)[^'\"]+", r"\1[REDACTED]", curl_cmd)

    # 2. Remove the Cookie header
    curl_cmd = re.sub(r"--header\s+['\"]Cookie:[^'\"]+['\"]\s*\\?\n?", "", curl_cmd)

    # Clean up any lingering empty lines
    curl_cmd = "\n".join([line for line in curl_cmd.splitlines() if line.strip()])

    return curl_cmd

def copy_to_clipboard(text):
    p = subprocess.Popen(['pbcopy'], stdin=subprocess.PIPE)
    p.communicate(input=text.encode('utf-8'))

if __name__ == "__main__":
    input_text = ""

    if not sys.stdin.isatty():
        input_text = sys.stdin.read()
    elif len(sys.argv) > 1:
        input_text = sys.argv[1]
    else:
        print("Usage: cleancurl '<curl_command>'")
        print("   Or: pbpaste | cleancurl")
        sys.exit(1)

    if input_text.strip():
        cleaned_text = clean_curl(input_text)
        copy_to_clipboard(cleaned_text)
        print("✅ Cleaned curl command copied to your clipboard!")
    else:
        print("❌ No input provided.")
```
